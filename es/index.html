FAQ
Preguntas frecuentes (profesionales)

¿Cómo se gestiona el servicio?

Cada solicitud se revisa según archivo, referencia y servicio solicitado. Cuando aplica, los datos fiscales o de empresa se recogen durante el proceso.

¿Cómo se confirma compatibilidad?

Por archivo original + referencia HW/SW + software + aplicación concreta.

¿Formatos aceptados?

Bin/ori y formatos nativos de herramientas profesionales. Enviar sin modificar.

¿Soporte post-escritura?

Sí. Soporte técnico para incidencias relacionadas con el archivo.
